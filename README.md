# Customer Churn Prediction

## Project Overview
Predict customer churn using machine learning classification models. This project demonstrates data preprocessing, feature engineering, and model comparison skills.

## Dataset
Kaggle Telco Customer Churn

## Models Used
- Logistic Regression
- Random Forest


## Results
- Best accuracy: [80.90%]


## Technologies
- Python
- Pandas, Scikit-learn, Matplotlib
- Jupyter Notebook

## How to Run
1. Clone this repository
2. Install requirements: `pip install pandas scikit-learn matplotlib seaborn`
3. Open `Customer Churn Prediction (1).ipynb` in Jupyter Notebook
4. Run all cells
